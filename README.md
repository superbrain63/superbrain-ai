
# SuperBrain AI

SuperBrain AI is a multi-skill AI assistant for chatting, resumes, content writing, email writing, and coding help.

Built with **OpenAI + Streamlit**.

### Features
- General Chat
- Resume & Cover Letter Writer
- Blog & Social Post Writer
- Email Writer
- Code Helper
- User Access Limit (Free & Premium mode)
- Razorpay Payments (coming soon)

### Installation
```bash
pip install -r requirements.txt
streamlit run app.py
